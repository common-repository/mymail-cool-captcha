Adds a Cool Captcha to your MyMail subscription forms

== Installation ==

1. Upload the entire `mymail-recaptcha` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings => Newsletter => Cool Captcha and configure the plugin
